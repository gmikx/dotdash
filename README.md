# DotDash

A Morse code learning app for iOS and Android, built with Flutter.

Practice sending and receiving Morse code - one letter at a time, or full sentences - with audio and haptic feedback.

---

## Features

- Send mode - tap out Morse code and get instant feedback
- Receive mode - listen to and decode Morse signals
- Adjustable speed (WPM) and tone frequency
- Audio feedback with configurable pitch
- Haptic feedback (device permitting)
- Dark and light theme

## Getting Started

### Prerequisites

- Flutter SDK `>=3.10`
- Xcode (for iOS)
- Android Studio or a connected Android device

### Running locally

```bash
git clone https://github.com/gmikx/dotdash.git
cd dotdash
flutter pub get
flutter run
```
